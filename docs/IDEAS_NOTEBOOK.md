# Ideas Notebook

A no-pressure running list for future project ideas. No structure required. No commitments made here. Getting ideas out of your head and into this file is the only goal.

## Rules for This File
- Write the date, the idea, and one sentence. That is enough.
- Do not plan, scope, or estimate ideas in this file. That comes later, in a future session, when the time is right.
- Nothing here becomes a real task until Sticker Quest ships Kindergarten to Libby.
- Review this file at the start of a new project — it is your backlog.

## Format

```
**[Date] — [Idea title]**
[One sentence describing the concept.]
```
